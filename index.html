Descubra o Número

Regras

  1. O computador escolhe um número secreto aleatório entre 1 e 100:
    • Esse número é mantido em segredo até o jogador acertar (ou perder).
  2. O jogador tem 10 tentativas para descobrir o número.
    • A cada palpite, o contador de tentativas diminui em 1.
    • Quando o contador chega a 0, o jogador perde o jogo.
  3. A cada tentativa, o jogo dá uma dica:
    • Se o palpite for menor que o número secreto → mostra “Seu palpite foi menor”.
    • Se o palpite for maior → mostra “Seu palpite foi maior”.
  4. Se o jogador acertar, o jogo mostra:
    • “Parabéns! Você acertou o número!”
    • O número secreto e o total de tentativas usadas.
  5. Se o jogador errar todas as tentativas:
    • O jogo exibe “Suas tentativas acabaram!”
    • Mostra também qual era o número secreto.
  6. Após o fim do jogo (acerto ou erro):
    • O jogador pode clicar em “Reiniciar” para começar uma nova partida.
    • Um novo número secreto será sorteado e as tentativas serão resetadas.
  7. Melhorias futuras (opcionais):
    • Mostrar o histórico de palpites anteriores.
    • Permitir escolher o intervalo (por exemplo, de 1 a 50).
    • Adicionar níveis de dificuldade (Fácil, Médio, Difícil).
    • Exibir mensagens coloridas (verde para acerto, vermelho para erro).
